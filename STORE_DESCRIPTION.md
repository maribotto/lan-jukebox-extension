### üöÄ How to Use

1. **Set Up Your Jukebox Server**
   - Download and run the LAN Jukebox server (find it on GitHub)
   - Note your server's IP address and port

2. **Configure the Extension**
   - Install this extension
   - Open any YouTube video
   - Click the extension icon
   - Enter your server address (e.g., `192.168.50.200:3000`)
   - Add username/password if your jukebox requires authentication
   - Click Save

3. **Start Adding Videos!**
   - Browse YouTube as normal
   - When you find a video to add, click the extension icon
   - Click "Add to Playlist"
   - Done! The video is now in the queue
