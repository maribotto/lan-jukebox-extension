# Store Assets for Chrome Web Store

This directory contains all promotional and marketing assets for publishing to Chrome Web Store.

## Screenshots (1280x800)

Required for Chrome Web Store listing. Upload all 3:

1. **screenshot-1.png** - Main popup interface showing video info and add button
2. **screenshot-2.png** - Success state with confirmation message
3. **screenshot-3.png** - Authentication view with username/password fields

## Promotional Tiles

### Small Tile (440x280)
**File:** `promo-tile-small.png`
- Used in search results and category pages
- Features play button icon and extension name

### Marquee Tile (1400x560)
**File:** `promo-tile-marquee.png`
- Featured placement tile (if selected by Chrome Web Store)
- Larger format with call-to-action button

## Upload Instructions

When submitting to Chrome Web Store:

1. **Screenshots section:**
   - Upload all 3 screenshots in order
   - They will be displayed in a carousel on the store page

2. **Promotional images section:**
   - Upload small tile (440x280)
   - Upload marquee tile (1400x560)
   - Store icon is already in the extension (128x128)

## Design Guidelines

All assets follow Chrome Web Store design guidelines:
- High resolution PNG format
- Clean, modern dark theme
- Consistent branding with extension UI
- Professional typography
- High contrast for accessibility

## Color Palette

Matches extension UI:
- Background: `#0f0f0f`
- Cards: `#1c1c1c`
- Primary accent: `#3b82f6`
- Text: `#ffffff`, `#b0b0b0`
